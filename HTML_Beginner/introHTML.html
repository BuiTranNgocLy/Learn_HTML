Giới thiệu HTML

HTML là gì?

    • HTML viết tắt của Hyper Text Markup Language

    • HTML là ngôn ngữ tiêu chuẩn tạo trang Web

    • HTML mô tả cấu trúc của một trang Web

    • HTML bao gồm một loạt các phần tử

    • Các phần tử HTML cho trình duyệt biết các hiển thị nội dung

    • Các phần tử HTML gắn nhãn các phần nội dung như: "this is heading", "this is a paragraph", "this is a link", etc.

Một tài liệu HTML đơn giản

Giải thích ví dụ

    • !DOCTYPE html: xác định đây là một tài liệu HTML 5

    • html: phần tử gốc của một trang HTML

    • head: chứa thông tin meta về treng HTML

    • title: tiêu đề cho trang HTML( hiển thị trong thanh tiêu đề của trình duyệt

    • body: phần thân của tài liệu, là vùng chứa tất cả nội dung hiển thị

    • h1: phần tử xác định tiêu đề lớn

    • p: phần tử xác định 1 đoạn văn

Phần tử HTML là gì?

    • Phẩn tử HTML là tất cả mọi thứ từ thẻ bắt đầu đến thẻ kết thúc
